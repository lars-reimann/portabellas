# IO

Sources:
- `src/portabellas/io/_table_reader.py` — `TableReader`
- `src/portabellas/io/_table_writer.py` — `TableWriter`

---

## TableReader

All methods are static. Accessed via `Table.read`.

### Static Methods

`csv_file(path: str | Path, *, separator: str = ",") -> Table` — Read a table from a CSV file.

`json_file(path: str | Path) -> Table` — Read a table from a JSON file.

`jsonl_file(path: str | Path) -> Table` — Read a table from a JSONL (JSON Lines) file.

`parquet_file(path: str | Path) -> Table` — Read a table from a Parquet file.

---

## TableWriter

Accessed via `table.write`.

### Constructor

`__init__(self, table: Table) -> None`

### Methods

`csv_file(self, path: str | Path) -> None` — Write the table to a CSV file.

`json_file(self, path: str | Path) -> None` — Write the table to a JSON file.

`jsonl_file(self, path: str | Path) -> None` — Write the table to a JSONL (JSON Lines) file.

`parquet_file(self, path: str | Path) -> None` — Write the table to a Parquet file.
